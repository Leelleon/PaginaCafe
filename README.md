# PaginaCafe

Proyecto de página web con Angular. 

:sparkles::sparkles: Bienvenidos a todos los que esten leyendo esto :sparkles::sparkles:

Hace algunos meses descubrí una nueva cosa que me apasiona, esta nueva pasión es el glorioso café :coffee:.

Esto nace por un reto de marca personal que me estoy proponiendo, más el poder poner en práctica nuevos conocimientos de desarrollo web.

*Los planes para esta página son:*
1. Tener información útil sobre el café y sus tantos tipos de preparación
2. Ofrecer un servicio similar al que ya ofrecen otras opciones como Brew Timer [Página de Google Play](https://play.google.com/store/apps/details?id=com.apptivity.brewtimer&hl=es&gl=US)
3. Formularios digitales para compartir tu propia cata de café [Hoja de ejemplo](https://cdn.shopify.com/s/files/1/1750/6127/files/boot_form_snapshot.PNG?16718227726967878276)
4. Usuarios para poder almacenar recetas y catas en la página
5. :zap:El cielo es el límite:zap:
